# Мини версия Heartstone

## Установка + загрузка зависимостей

```bash
npm install
```

## dev сборка

Будет находиться на `http://localhost:3000`:

```bash
npm run dev
```

## Билд

```bash
npm run build
```

Превью:

```bash
npm run preview
```

Check out the [Документация по NNxt3](https://nuxt.com/docs/getting-started/deployment) for more information.
